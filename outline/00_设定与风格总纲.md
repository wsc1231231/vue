# 00. 设定与风格总纲

## 一、作品定位

- **类型:** 都市 / 校园 / 青春 / 多女主 / 逆袭系统 / 半擦边爽文
- **目标篇幅:** 380-400 章, 每章 2800-3500 字
- **核心卖点:**
  1. 多女主群像自然铺开, 不强行人均恋爱, 各女主独立成长线
  2. 地道中式小地级市高中青春 (烟火气 / 老街 / 县城关系网 / 月考卷子)
  3. 未来自我 "回响" 式系统 (非面板, 更偏梦境闪回+信)
  4. 爽文打脸节奏 (升班 / 月考 / 大联考 / 对虚伪社交圈冷处理)
  5. 抽象网感文风 + 偶尔贫嘴 + 关键时刻不怂
- **写作视角:** 以男主陆辞为绝对第一主角的第三人称限知; 每卷允许 1-2 段女主心理回闪 (不超一千字)
- **情感基调:** 开放式不明归宿; 每个女生的情感都真诚、独立、有自己的生活, 不为男主而存在
- **爽感来源:** ① 知识 "回响" 让男主学科起飞 ② 对虚伪社交圈的冷处理 ③ 家庭经济改善 ④ 女主群像自然亲近 ⑤ 反派被现实规则打脸

## 二、时代与地理

**时代背景:** 架空近未来, 按 2023-2024 年代氛围书写。
- 智能手机普及; 短视频/小红书/B站/微信流行
- 高中生标配: 手机一台, 但学校管理严格, 课上不许玩
- 移动支付为主, 现金很少

**主要地理:** 临川市 (虚构中部地级市, 位于长江中游以南, 有江有山, 人口约 300 万)
- 下辖 4 区 3 县, 男主一家住在市辖区 **青枫区** 老城
- 青枫区特征: 老旧自建房 / 菜市场 / 窄巷纵横 / 烟火气十足 / 中低收入聚居
- 市区核心: **滨江新区** (富人区, 多女主家住此, 高端小区 + 国际学校分校所在地)
- 城南: **长潭湖风景区** (5A景区, 后期校游 / 约会 / 秋游场)
- 东郊: **青山镇** (男主老家, 有爷爷奶奶, 寒假回乡用)

**主场景 — 临川市第一中学 (临一中):**
- 全市顶尖重点高中, 分班制:
  - **零班** 1个 (省理科竞赛班, 清北华五候选)
  - **奥赛班** 5个 (理3文2, 一本保底, 985冲刺)
  - **平行班** 8个 (重本率约 50%)
  - **重点帮扶班** 3个 (专科 + 二本边缘)
- 男主 **陆辞** 初始平行八班 (理科), 第一次月考后升入 **奥赛三班 (理)**
- 青梅 **苏知予** 始终奥赛二班 (理科), 与奥赛三班相邻 (同三楼)
- 六层主教学楼; 奥赛班在三楼, 平行班分四至六楼, 重点帮扶班在一二楼 (便于教导处管)
- 学校东门外: 老街小吃 / 网吧 / 奶茶店 / 小卖部 (校园社交核心)
- 学校西门外: 新建商业街 (有书店、眼镜店、电影院)

**周边城市地标:**
- 市中心 **河坊街** (步行街, 约会圣地)
- **临川市图书馆** (情感高光场景: 雨天相遇)
- **星麓 livehouse** (后期文艺线 / 苏知予古筝演出)
- **临川大学** (本地一本, 多女主有人最终在此)

## 三、世界规则

**1. 普通人的天花板**
   - 纯靠自己努力高考 → 双非一本, 上限 211 尾部
   - 有好师资 + 系统辅助 → 985/211 上游
   - 有家庭关系 → 保送 / 自招 / 人才引进
   - 规则残酷但不绝对, 男主要靠系统打破

**2. 社交货币化**
   - 县城关系网盘根错节, 家长职务能决定命运
   - 陆家底层, 多位女主家为本地显贵
   - 这种反差是本书爽感与冲突来源

**3. 系统的现实伪装**
   - 奖励现金来自 **临心实验基金会** (虚构, 但在本书世界中为合法注册机构)
   - 伪装为 "青年心理状态长期追踪项目" 的补贴
   - 男主是"唯一受试者", 走正规财务流程, 可过税务审计
   - 老陆(父亲) 查过, 发现基金会存在且正规, 遂放心

## 五、创作基调

- **文风:** 短句 / 快节奏 / 网感 / 嘴硬心软的内心独白 / 反差吐槽
- **人物:** 男主嘴毒但不恶毒, 女主各有性格, 反派坏但不恶心到读不下去
- **场景:** 小县城写实, 菜市场 / 食堂 / 操场 / 晚自习灯光 / 老旧楼道这些都要写出 "烟火气"
- **情感:** 点到为止, 最怕用力过猛; 该冷就冷, 该爆发就爆发, 绝不滥用 "眼眶一热"
