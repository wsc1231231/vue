# 00. 设定与风格总纲

## 一、作品定位

- **类型:** 都市 / 校园 / 青春 / 多女主 / 逆袭系统 / 半擦边爽文
- **目标篇幅:** 400-410 章, 每章 2800-3500 字
- **核心卖点:**
  1. 五位女主 + 一位青梅, 各自独立成长线, 不人均恋爱
  2. 地道中式小城市高中日常 (老街 / 菜市场 / 奶茶店 / 网吧 / 晚自习)
  3. "未读短信" 式系统, 低调不喧宾夺主
  4. 爽文打脸节奏 (月考爆种 / 打篮球救场 / 对虚伪小团体冷处理)
  5. 抽象网感文风 + 嘴硬心软内心独白 + 偶尔贫嘴
- **视角:** 男主陆辞第三人称限知为主; 每卷允许 1-2 段女主心理回闪 (不超一千字)
- **情感基调:** 开放式归宿; 每个女生的情感都真诚独立, 不为男主而存在
- **爽感来源:** ① 系统道具让学科起飞 ② 挣钱改善家里 ③ 对虚伪社交圈的冷处理 ④ 多女主群像自然亲近 ⑤ 篮球场 / 摄影圈的出风头瞬间

## 二、时代与地理

**时代背景:** 架空近未来, 按 2023-2024 年代氛围书写。
- 智能手机 / 短视频 / 微信 / 支付宝 / B站 / 小红书
- 高中校园管理严格, 课上不许玩手机, 晚自习三节
- 移动支付为主, 纸币极少见

**主地点:** 南安市 (虚构东南沿海中型地级市, 人口约 250 万)
- 下辖 3 区 2 县, 男主一家住在市辖区 **西城区**
- 西城区特征: 老旧自建房 / 烟火菜市场 / 巷子纵横 / 五金小吃街
- 新区: **海屿新区** (富人区, 多位女主家住此, 高端商业综合体)
- 南郊: **长屿风景区** (后期校游 / 毕业约拍场)
- 东北方向: **青田村** (男主父亲老家, 过年回乡用)

**主场景 — 南安市第三中学 (南安三中):**
- 市内三所顶尖重点之一, 分班制:
  - **竞赛班** 1 个 (省赛保送路线)
  - **奥赛班** 5 个 (理 3 文 2, 985 冲刺)
  - **平行班** 8 个 (重本率 40-55%)
  - **帮扶班** 3 个 (专科 + 二本边缘)
- 男主 **陆辞** 始终在 **高三 (7) 班 (平行理科班)**。本书主动不走升班路线——男主选择留在原班
- 青梅 **苏知予** 始终在 **高三 (2) 班 (奥赛理科二班)**, 与陆辞同一栋教学楼不同楼层
- 主教学楼六层: 竞赛班三楼, 奥赛三四楼, 平行班五六楼, 帮扶班一二楼
- 学校西门: 老街小吃 / 奶茶 / 网吧 / 小卖部 (校园社交中心)
- 学校东门: 新开的商业综合体 (电影院 / 书店 / 眼镜店)

**周边地标:**
- 市中心 **鼓楼步行街** (约会圣地)
- **南安市立图书馆** (雨天相遇场)
- **叁月咖啡** (陆辞父亲五金店对面咖啡店, 学姐祁漪兼职处)
- **南安师范大学** (本地一本, 多位女主高考归宿选项之一)
- **西城网咖** (学妹白知活动地)

## 三、世界规则

**1. 普通人的天花板**
   - 纯靠自己努力高考 → 双非一本, 上限 211 尾部
   - 有好师资 + 系统辅助 → 985/211 上游
   - 有家庭关系 → 保送 / 自招 / 人才引进
   - 规则残酷但不绝对

**2. 社交货币化**
   - 本地关系网盘根错节
   - 陆家底层, 部分女主家境优越
   - 这种反差是爽感与冲突来源之一, 但不构成苦大仇深的底色

**3. 系统的现实伪装**
   - 奖励现金通过 "匿名投资人" 标签支付宝转入
   - 男主查过无任何公司名, 只有一段备注文字
   - 老陆 (父亲) 曾怀疑过, 男主编造说是 "网上接单做图的零散收入", 搪塞过去
   - 数额控制: 单笔不超 4999, 避免触发银行风控

## 五、情感线节奏总纲

- **青梅苏知予**: 全书绝对女主线, 从互怼 → 默默观察 → 心跳 → 不敢承认 → 毕业后她没告白, 留下一本六年日记
- **应采禾 (班花前桌)**: 自然熟稔 → 学习互助 → 小心动 → 毕业去外省上大学
- **姜酌 (文学社社长)**: 雨天图书馆偶遇 → 文字交流 → 清冷文艺线 → 考去北京
- **白知 (网吧学妹)**: 外校意外相识 → 双排吃鸡 → 反差萌 → 走艺考
- **祁漪 (咖啡店大三学姐)**: 姐弟式关心 → 五金店邻居般亲近 → 她毕业后留南安工作
- **聂恬 (朋友之妹)**: 家庭饭局认识 → 偶尔拌嘴 → 暗暗在意 → 继续高二高三

## 六、创作基调

- **文风:** 短句 / 快节奏 / 网感 / 嘴硬心软的内心独白 / 反差吐槽
- **人物:** 男主嘴毒但不恶毒, 女主各有性格, 对手坏但不恶心
- **场景:** 小城市写实, 菜市场 / 食堂 / 操场 / 晚自习灯光 / 老旧楼道要写出烟火气
- **情感:** 点到为止; 该冷就冷, 该爆发就爆发, 绝不滥用 "眼眶一热"
- **系统任务呈现格式:**

```
凌晨 03:38, 手机亮了一下。
"陆辞" 给 "陆辞" 发来一条短信:

【(一句勾起遗憾的生活场景描写)。
任务: (具体行动要求)。
奖励: (金额 / 道具 / 技能碎片)。】
```

文案基调是 "十八岁的自己看完会愣一下" 的那种。
