# 05. 风格语言工具包

下游 AI 使用本文档时, 把它作为 "语言与节奏字典"。

## 一、句式特征

### 句长
- 以短句为主, 平均 10-18 字
- 长句用于描写场景, 不连用两个以上
- 对话尽量短, 三五字一句

### 节奏
- 每段不超过 5 行
- 对话与描写穿插, 不整段描写
- 同一段内避免重复用 "他/她"

### 视角切换
- 主视角陆辞第三人称限知
- 每卷允许 1-2 段女主视角 (不超过 1000 字), 用斜体或标题提示
- 绝不乱切视角, 切换必须有标记

## 二、男主语言工具箱

### 口头禅 (可反复用)
- "啧" / "啧啧" / "得了吧" / "我超" / "沃日" / "我谢谢你啊"
- "行吧" / "哎, 这日子" / "年轻人嘛, 有点想法很正常"
- 被怼时: "……你说的对。" (配表情: 眼神空洞)
- 装深沉被戳穿时: "咳, 我觉得你对自己有点太苛刻。"

### 内心 OS 句式
- "嘴上 A, 心里想 B" 的反差
- 例:
  - (嘴上) "这怎么吃得完" (心里) 再来一碗。
  - (嘴上) "我不饿" (心里) 肚子叫了第三次了。
  - (嘴上) "你挺好看的" (心里) 拍一张压箱底。
- 每章建议 5-8 处反差 OS

### 网络梗库 (可用但不滥用)
- "我超" / "沃日" / "我焯"
- "绷不住" / "嘴角压不住" / "嘴角扬起一个弧度"
- "河狸" / "寄" / "典" / "孝" / "麻"
- "社会工程学" / "爹系发言" / "下头男"
- "笑死" / "笑拥" / "绷"
- **使用原则: 每章不超过 3 个网络梗, 否则廉价。**

### 地道方言点缀 (低频使用)
- "搞什么搞" / "嗨呀" / "搞毛"
- 南方生活梗: "吃过饭了吗?" 代替 "吃了吗"
- 陆辞父亲说话偶尔带点本地口音

## 三、女主语言差异化

| 女主 | 口头禅 | 特征 |
|------|-------|------|
| 苏知予 | "滚啊" "有毛病吧" "你死" | 傲娇炸毛, 爆破前三秒是有铺垫的 |
| 应采禾 | "陆老板!" "过来!" "我跟你说!" | 爱叫人, 话多, 热情 |
| 姜酌 | "嗯" "哦" "谬赞了" | 惜字如金, 用典雅句式 |
| 白知 | "卧槽" "笑死" "你挂了?" | 一口一个网感脏话, 实际很干净 |
| 祁漪 | "来啦" "自己坐" "少年" | 温柔缓慢, 句尾有笑意 |
| 聂恬 | "陆辞你听着!" "凭什么啊!" | 语速快, 爆炸开口 |

**铁律: 每位女主开口的第一句, 读者不看名字也能认出来是谁。**

## 四、描写模板

### 场景起手式 (每章开头可任选)
1. "XX 点, 南安三中 XX 教室。" (时间+地点直入)
2. "阳光/暴雨/月色, XX 街/走廊/操场, 少年在 XX。" (环境+动作)
3. 人物对话直接起手, 不交代背景

### 人物初登场格式
- 1-3 行外貌 + 1 行衣着 + 1 句神态
- **不要大段写人物简历**, 分散到后续对话中

### 情感描写要点
- 避免 "他心里酸酸的" / "眼眶一热" 这种烂大街
- 用动作代替情感: "他把烟按灭在窗台, 没说话" / "她耳尖慢慢红了" / "他的手悬在门把上停了两秒"
- 用环境烘托: 雨 / 光 / 风 / 夜 / 台灯

### 擦边描写尺度
- **允许**: 描写校服下的身形轮廓 / 腿 / 脚踝 / 耳垂 / 颈侧 / 锁骨 / 晒红的脸
- **不允许**: 生殖器 / 具体性行为 / 强迫 / 未成年性描写
- **原则**: 暗示而非直写, 三笔带过

## 五、笑点设计

### 每章至少 1-2 个笑点, 优先级:
1. 男主内心 OS 与嘴上言行的反差
2. 抽象贫嘴的大段独白 (不超过 5 行)
3. 朋友 (唐越/乐衡) 的神来一句吐槽
4. 不合时宜的系统短信推送 (例如接吻前一秒弹出任务 "与某某进行一次深入交谈")

### 避免的笑点
- 物化女性 (外貌羞辱)
- 拿家庭困难开玩笑
- 拿死者开玩笑
- 地域歧视

## 六、章末钩子模板

每章必有钩子, 三种形式:
1. **悬念型**: "他刚想睡, 手机又亮了一次。这次不是凌晨 03:38。"
2. **暧昧型**: "少女的耳垂在灯下几乎要燃起来, 他本能地转开头。" (下章接续)
3. **预告型**: "第二天一早, 他并不知道, 这一天是改变所有人轨迹的一天。"

## 七、章名参考风格

学习原始作品的章节命名思路, 用短句或口语化表达, 例:
- 《你这是? 早餐?》
- 《不是, 哥, 她怎么也来了》
- 《凌晨三点的那条短信》
- 《对门宝宝, 别误会》
- 《这个分数, 老师的茶都凉了》
- 《体育老师哭了, 他哭什么》
- 《篮球场上的两秒钟》
- 《陆老板, 你笑什么?》

每章名 5-15 字, 口语化, 带一点 "让读者点进来" 的钩子感。

## 八、禁忌清单

1. **不写**: 具体性描写 / 未成年人性行为 / 严重暴力血腥 / 重大违法犯罪主线 / 师生恋实质发展
2. **谨慎**: 饮酒、抽烟 (男主父亲可以抽, 男主只在成年后试过一次)
3. **绝对避免**: 歧视 (地域/性别/家庭/外貌)、嘲讽精神病人、拿死亡开玩笑
4. **注意**: 每章校内场景要符合高三实际作息 (7:30 早读, 12:10 午饭, 17:30 晚饭, 21:40 放学, 23:00 熄灯)
