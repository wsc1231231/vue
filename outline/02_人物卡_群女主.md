# 02. 人物卡 — 群女主

本卷收录除青梅苏知予外的 5 位女性主要角色。按首次登场先后排列。

---

## 1. 应采禾 (Yīng Cǎihé) — 班花 / 前桌

### 基础档案
- **年龄**: 18
- **身高**: 166, 白净微胖, 笑起来眼睛弯成月牙
- **外貌**: 圆脸微肉, 皮肤白, 两颗小酒窝; 喜欢穿马卡龙色毛衣配牛仔裤; 会戴一副无框平光镜假装 "文静风", 但根本掩盖不住爱笑的属性
- **班级**: 南安三中高三 (7) 班, 陆辞同班, 高三开学调位后坐在陆辞前桌
- **家庭**:
  - 父 **应立群** (49, 西城区 "采禾书屋" 老板, 读书人, 话不多但有涵养)
  - 母 **柳月茹** (47, 西城区一中图书馆管理员)
  - 独生女, 家境小康, 家里一墙壁的书
  - 住西城区老城 **采禾书屋** 二楼 (自家房子)

### 性格核心
- **表层**: 开朗 / 爱笑 / 话多 / 热情 / 爱八卦
- **深层**:
  - 不傻, 知道谁在玩心眼; 只是不愿意掉面子去计较
  - 对陆辞有淡淡好感 (初中同学时就觉得他好看), 但表达方式是 "喊他过来帮忙"
  - 对家里书店有一种不宣之口的守护感
  - 高三压力大时会偷偷吃一整条巧克力, 谁看见就和谁翻脸
- **口头禅**:
  - "陆辞, 过来一下!" "你猜猜看?" "我跟你说!"
  - 被打趣时: "哎呀烦死了啦~"
- **怪癖**:
  - 喜欢看推理小说, 床头一排东野圭吾
  - 会 "给人起外号", 陆辞被她叫 "陆老板" 因为陆家开五金店
  - 每周五放学必去鼓楼街买双皮奶

### 技能 / 特长
- **学业**: 年级排名 120 左右 (全年级 1500+), 语文英语尚可, 数学物理拉胯
- **朗诵**: 学校运动会主持保留嘉宾, 声音脆亮
- **书店辅助**: 帮父母整理店务, 会简单的书籍分类与推荐
- **修图**: 会用手机修图软件, 给陆辞的照片做过调色 (水平有限但热情)

### 与男主的关系线
- **卷一**: 开学调位坐陆辞前桌, 一开始只是同学; 被陆辞帮过两次 (修文具袋拉链、一次篮球场边的冰可乐), 开始有好感
- **卷二**: 月考陆辞爆种, 她第一时间转过头瞪大眼睛: "陆老板, 你是不是偷偷报班了?"
- **卷三**: 运动会当陆辞篮球赛主持, 给他加了一整场的私货 bgm
- **卷四**: 生病请假几天, 陆辞送一次作业, 她爸妈第一次见到这个 "陆同学", 觉得这孩子不错
- **卷五**: 寒假给陆辞发 "我爸今天又进了一批书, 你要不要来看看", 陆辞去了, 买走一本《目送》
- **卷六**: 她开始主动约陆辞吃饭, 但每次都带着其他同学, 不是单独
- **卷七**: 模考后的某次放学, 她终于单独约他去喝双皮奶, 没告白, 但两人之间多了一种 "你懂我懂" 的默契
- **卷八**: 她考去了杭州某财经类二本, 毕业晚宴上哭了很久但没说原因
- **终章**: 大学开学后偶尔朋友圈点赞, 暑假各回南安时会约饭

### 关键写作提示
- **避免**: 把她写成 "绿茶" 或 "抢青梅"。她阳光自然, 对苏知予友善
- **关键意象**: 前桌转头的马尾 / 双皮奶 / 书店木楼梯 / 毛衣袖口露出的手腕
- **她的爽点**: 陆辞帮她修电脑 / 她偷偷夸陆辞摄影好看

## 2. 姜酌 (Jiāng Zhuó) — 文学社社长 / 清冷文艺型

### 基础档案
- **年龄**: 18
- **身高**: 168, 瘦, 锁骨明显, 气质清冷
- **外貌**: 细长眉, 薄嘴唇, 常年一头黑长直; 穿衣偏深色系, 白衬衫+深色长裙+帆布鞋; 很少笑, 笑起来很浅
- **班级**: 南安三中高三 (4) 班 (奥赛文二班)
- **家庭**:
  - 父 **姜砚之** (50, 南安三中退休前语文教研组长; 对姜酌教育宽松)
  - 母 **文时雨** (48, 市图书馆副馆长)
  - 独生女, 家境清贫但精神富足
  - 住市图书馆宿舍楼

### 性格核心
- **表层**: 冷淡 / 话少 / 不合群 / 思辨强 / 爱写作
- **深层**:
  - 对人观察细腻, 看穿不说穿
  - 对陆辞的第一印象: 这人嘴上贫, 眼睛里有东西
  - 不爱社交但珍视极少数朋友
  - 一旦开口说话, 句句能戳到人心
- **口头禅**:
  - "嗯。" "哦。" "你怎么看?"
  - 被夸时: "谬赞了。"
  - 发朋友圈常用诗词片段, 没人看得懂

### 技能 / 特长
- **写作**: 年级语文三甲钉子户, 个人公众号 "酌酌杯" 有两千粉
- **摄影**: 会用胶片拍风景, 和陆辞摄影线第一次正面交汇
- **古琴**: 会, 但不如青梅专业

### 与男主的关系线
- **卷一**: 未出场
- **卷二**: 校报征稿活动初次见陆辞 (陆辞以摄影身份投稿), 被她的文字吸引但未深入
- **卷三**: 文学社+摄影社联谊周, 两人正式搭档产出 "图文合集", 合作顺利
- **卷四**: 某个图书馆雨天, 两人意外共用一把伞回校, 一路无话但不尴尬
- **卷五**: 寒假她给陆辞寄了一本签名诗集 (她出过一本小册子), 落款 "赠陆辞, 愿你的快门永远温柔"
- **卷六**: 两人合作的一个 "南安街头人物志" 系列在市文艺圈小火, 她邀陆辞一起去市文联见人
- **卷七**: 志愿她坚定选北师大中文, 鼓励陆辞去北京读同济; 两人在一次咖啡店聊到凌晨
- **卷八**: 她考上北师大中文, 毕业晚宴上两人有一段很短的对话: "我不会等你, 但我会一直读你拍的照片。"
- **终章**: 大学后她偶尔给陆辞寄明信片, 没有落款

### 关键写作提示
- **避免**: 写成忧郁文艺病; 她底色是明白人, 不无病呻吟
- **关键意象**: 白衬衫 / 帆布鞋 / 图书馆的灰尘光线 / 签名诗集 / 明信片
- **她的爽点**: 她给陆辞念自己写的一段小说片段, 陆辞听懂并回了一句对仗

## 3. 白知 (Bái Zhī) — 网吧学妹 / 网瘾反差

### 基础档案
- **年龄**: 16 (高一)
- **身高**: 158, 小只, 瘦
- **外貌**: 黑色齐刘海, 圆脸, 眼睛大而圆像小猫; 常年连帽卫衣+牛仔短裤+AJ鞋; 左耳打三个耳洞, 平时戴银色小环
- **班级**: 南安三中高一 (11) 班 (普通班, 成绩在班内前三)
- **家庭**:
  - 父 (离异后不联系)
  - 母 **骆晚笙** (38, 酒吧驻场 DJ, 作息颠倒, 女儿实际独立生活)
  - 住海屿新区一套小公寓 (母亲买的)

### 性格核心
- **表层**: 痞 / 酷 / 不爱笑 / 一口一个 "卧槽"
- **深层**:
  - 早熟, 对生活有超出年龄的清醒
  - 对母亲爱恨交织, 希望母亲过得好但也埋怨她常年不在
  - 对陆辞第一次产生 "这人不嫌我" 的安全感
  - 外冷内烫, 认人后极度忠诚
- **口头禅**:
  - "卧槽" "笑死" "nm" "你挂了?"
  - 认真时: "……别担心, 我挺好的。"

### 技能 / 特长
- **游戏**: 无畏契约王牌段位, 手速一流
- **学业**: 年级前三十, 文理科均衡, 英语出奇好
- **街舞**: 初中练过, 会 lock
- **调酒**: 跟母亲学过一点, 会调三四款简单酒 (只给自己喝, 但她不喝醉)

### 与男主的关系线
- **卷三**: 某周六陆辞在西城网咖包夜, 邻桌就是她, 双排吃鸡被她带飞, 加微信
- **卷四**: 她母亲生病住院, 陆辞陪她去过一次医院; 她第一次对陆辞说 "谢谢"
- **卷五**: 寒假她独自过年, 陆辞邀她来陆家吃年夜饭, 被陆父陆母当自家孩子招待
- **卷六**: 她开始经常出现在陆辞生活边缘 (五金店门口送外卖 / 咖啡店偶遇 / 放学骑电动到校门口)
- **卷七**: 她高一下学期成绩直线拉升, 因 "陆哥说过好好读书以后自由", 她也要自由
- **卷八**: 陆辞高考期间, 她每晚给他发一句 "你加油, 我看着呢"
- **终章**: 两年后她可能考南安三中的竞赛班, 或者走艺考去北京; 留白

### 关键写作提示
- **避免**: 写成 "坏女孩"; 她只是早熟, 不是堕落
- **关键意象**: 连帽卫衣 / AJ / 网咖机械键盘声 / 银色耳环 / 凌晨的公交车
- **她的爽点**: 陆辞替她教训某个在网咖骚扰的醉汉

## 4. 祁漪 (Qí Yī) — 咖啡店学姐 / 姐弟式暧昧

- **身高**: 170, 纤瘦, 气场成熟
- **外貌**: 长发低马尾, 长睫毛, 唇色偏淡; 穿衣素雅, 白 T + 工装裤 + 帆布鞋就能撑起气场
- **身份**: **叁月咖啡** 店员 (陆辞父亲五金店对面, 下午到晚上班)
- **家庭**:
  - 父 (出车祸去世, 祁漪上小学六年级那年)
  - 母 **苏念音** (46, 原在国企退休, 现与祁漪同住一小公寓)
  - 独生女, 一边读书一边打工补贴家用

### 性格核心
- **表层**: 温柔 / 话慢 / 目光稳 / 笑容节制
- **深层**:
  - 坚韧到心疼: 父亲去世后母亲抑郁几年, 她是家里的支柱

  - 清楚界限, 不允许自己越界
- **口头禅**:

  - 对陆辞: "少年, 你今天又在看什么?"

### 技能 / 特长

- **茶艺**: 喜欢, 自己在家泡茶
- **骑行**: 会骑折叠车, 周末常去长屿景区

### 与男主的关系线
- **卷一**: 出场一次, 陆辞去给父亲送午饭时路过咖啡店, 两人眼神一对, 她对他笑了一下
- **卷三**: 陆辞开始频繁去咖啡店蹭 wifi 改作业, 她记住他不加糖
- **卷四**: 她母亲病重住院缺药钱, 陆辞无意中得知后, 借系统奖励存款悄悄垫付了一次 (以父亲邻居身份), 后被她知道, 她第一次鞠躬感谢
- **卷五**: 寒假她独自值班, 陆辞陪坐了一下午, 两人聊天多但界限清楚
- **卷六**: 她出国交换半学期, 回来时给陆辞带了一本日本摄影集 (森山大道)
- **卷七**: 毕业在即, 她决定留在南安本地考当地事业单位
- **卷八**: 陆辞高考那几天, 她在咖啡店给他留了一杯冰美式
- **终章**: 她留南安工作, 继续做陆辞的 "街坊姐姐"; 界限始终在

- **必须守住**: 她与陆辞不发生恋爱关系, 哪怕最暧昧的瞬间也要克制
- **关键意象**: 叁月咖啡的黄灯 / 桂花酸奶 / 折叠车 / 森山大道写真集 / 淡色口红印在杯沿
- **她的爽点**: 她在陆辞面前第一次哭 (因为母亲病情) 被陆辞默默递纸巾而非安慰
