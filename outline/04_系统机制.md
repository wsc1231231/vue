# 04. 系统机制 — "未读短信" 

## 一、系统名称与包装

**名称:** 本书中系统无正式名字, 男主自称 "凌晨短信" 或 "我那个短信"。
**外观:** 普通手机短信界面, 发件人是陆辞自己的手机号 (187-xxxx-1812), 内容为任务文字, 有时附图附链接。
**触发时间:** 每天凌晨 **03:38** 前后 (有时略早略晚), 手机屏幕亮一次, 收到 1-3 条短信。
**例外机制:** 剧烈情感波动 (父亲病倒 / 被公开羞辱 / 久别重逢) 时可能临时推送。
**持续性:** 全书一直存在, 终章最后一章留有 "短信停了很久很久然后又亮了一次" 的开放结尾。

## 二、任务分类

| 类别 | 说明 | 出现频次 | 例 |
|------|------|---------|------|
| 主线任务 | 与每卷主题深度绑定 | 每卷 3-5 个 | 月考进班级前十、运动会打完全场 |
| 日常任务 | 围绕 "弥补遗憾", 偏向生活细节 | 每 3-5 章 1 个 | 给妈妈买一双运动鞋、陪爸爸吃一次夜宵 |
| 挑战任务 | 考验男主极限 | 每卷 1-2 个 | 一口气跑五公里、连续一周不玩手机 |
| 隐藏任务 | 不主动提示, 男主碰巧触发 | 每卷 0-1 个 | 雨夜偶遇某人并送伞回家 |

## 三、奖励体系

### 1. 现金 (通过支付宝转账)
- 发件标签: "匿名投资人奖励"
- 单笔范围: 100 - 4999 元 (避免银行风控)
- 全书总流入: 预估 40 万 - 60 万
- 主要用途:
  - 给家里换电器 / 换手机 / 换衣服鞋
  - 摄影器材升级
  - 偶尔给朋友买礼物

### 2. 道具 (从短信链接下载后 "实体化")

| 道具名 | 功能 | 出现卷 |
|--------|------|--------|
| 记忆纸条 (薄荷味) | 贴眉心 5 小时, 该时段看过的书能过目不忘 | 卷一 |
| 清醒糖 | 吃一颗保持 8 小时清醒不疲劳, 每天限用 1 颗 | 卷二 |
| 暖手贴 (自热) | 冬天暖一整天, 可以用来送人 | 卷四 |
| 专注耳塞 | 戴上后 2 小时屏蔽一切外界声音 | 卷六 |
| 隐形洗护油 | 擦运动服瞬间无汗味, 篮球赛场景专用 | 卷三 |
| 灵感卡片 | 看一眼能灵光一闪, 摄影构图思路涌现 | 卷四 |
| 底片盒 | 保存一份 "未来画面" 给男主自己看, 每卷限 1 次 | 卷二起 |

### 3. 技能碎片
- 通过完成特定类别任务获得
- 每 5 个碎片可兑换一级技能提升
- 可兑换: 篮球 (投篮精度/运球/爆发)、摄影 (构图/后期/光感)、语言 (口才/应变)、生活 (烹饪/修理/驾驶)

## 四、任务呈现格式 (下游 AI 写作模板)

**格式 A — 标准主线:**
```
凌晨 03:38, 陆辞的手机屏幕亮了一下。

"陆辞" 给 "陆辞" 发来一条短信:

【(一句勾起遗憾的生活场景描写, 40-80 字)。

任务: (具体行动要求, 一句话, 30-50 字)。

奖励: ¥xxxx 元 / (道具名) / (技能碎片 x N)。】
```

**格式 B — 隐藏触发:**
```
走到半路, 陆辞手机突然震了一下。他以为是日常消息, 点开一看 ——

【(无序言, 直接一行场景): ……
任务: ……
奖励: ……】
```

**格式 C — 附图短信:**
```
短信底部多了一张低分辨率截图。图里是 (未来某场景的模糊描述), 陆辞看了很久, 没说话。
```

## 五、系统节奏铁律

1. **每卷开头 1-2 章必须推进一个主线任务**, 让读者知道这卷的方向
2. **卷末必须结一个主线**, 给读者满足感
3. **每章最多出现 1 条系统短信**, 不要滥用
4. **短信文案不能千篇一律**, 每条都要 "读了会愣神" 的那种写法
5. **奖励数额与任务难度挂钩**, 不可反向 (太大的钱给太小的事)
6. **道具不能 OP**, 每个道具都有使用次数/时间限制
7. **不推送 "追某某" 类任务**, 感情的事系统不插手; 可以推送 "送一把伞" 这类动作, 但不能指明对象

## 六、系统的 "未来自我" 语气模板

- 不带情绪的陈述: "三十八岁的你习惯在医院走廊抽烟。"
- 带一点怅然: "那年你本可以多陪她一会, 你没有。"
- 偶尔自嘲: "你那时以为这条路走得对, 其实没有。"
- 极少关切: "别熬夜了, 少年。"

## 七、系统被发现的风险与遮掩

- 陆辞早期尝试问 "你到底是谁", 短信不回复
- 父亲一度问钱哪来, 陆辞编造 "网上接零散修图单", 拿出一套截图蒙混过关
- 中期陆辞开始不解释, 家里也默认他 "有自己的挣钱渠道"
- 终章陆辞没公开过系统存在, 包括对青梅
