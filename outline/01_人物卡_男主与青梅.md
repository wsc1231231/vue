# 01. 人物卡 — 男主与青梅

## 男主:陆辞 (Lù Cí)

### 基础档案
- **年龄**: 18
- **身高**: 182 cm, 瘦而有肩线, 打篮球锻炼出来的身材
- **外貌**: 眉骨略高, 眼皮薄, 下颌线清晰; 笑起来有点邪气, 不笑时有点凶。黑发偏硬, 刘海常扫过眉毛; 耳垂小, 容易晒红
- **班级**: 南安三中高三 (7) 班 (普通理科班)。**全书不升班, 这是男主主动选择**
- **家庭**:
  - 父 **陆卫东** (45, 西城区"和兴五金建材行"老板, 性子慢, 爱抽烟, 话少)
  - 母 **许秀兰** (43, 南安市西城区人民医院普外科护士, 刀子嘴豆腐心, 唠叨)
  - 家境: 2021 年爷爷重病花光家底, 至今刚还完欠债, 月结余不多
  - 一家三口, 住西城区 **安河小区** 8 栋 5 楼 502; 对门 501 是苏家

### 性格核心
- **表层**: 吊儿郎当 / 嘴毒 / 懒散 / 对学习无所谓 / 对陌生人友善但带距离
- **深层**:
  - 对父母孝顺, 会替母亲拎菜, 记得父亲鞋码
  - 对虚伪敏感, 被踩一次就永不回圈
  - 看似冷漠, 共情力其实很强
  - 务实, 信 "先把自己温饱解决了再谈理想"
  - 对女生不主动, 但一旦接纳就护短
- **口头禅**:
  - "啧" "得了吧" "我超" "沃日"
  - "……行吧" "我谢谢你啊"
  - 被戳破装深沉时: "年轻人嘛, 有点想法很正常"
  - 内心 OS 和嘴上常常撕裂, 嘴上一回事心里一回事
- **怪癖**:
  - 不喝甜豆浆
  - 饭必须热
  - 睡前手机放伸手够不到的地方防熬夜
  - 对小孩有耐心, 对大人没耐心

### 技能 / 特长
- **摄影**: 初二用压岁钱攒了一台二手富士 X-T10, 高中一直在拍。擅长人像与街头扫街, 豆瓣账号 "光之弃子" 不到百粉, 风格偏阴翳系, 后期逐渐转暖
- **篮球**: 校队替补主力水平, 打得分后卫, 三分投得好, 关键球心脏大; 曾一学期连续晨跑五公里
- **动手**: 帮父亲看店学的, 家电小毛病自己修
- **做饭**: 基础家常能做出来, 不是立人设点

### 未来线 (三十八岁的陆辞)
- 专科毕业后干过工厂质检 / 外卖 / 摄影助理 / 短视频剪辑
- 28 岁相亲结婚, 32 岁创业摄影工作室亏 78 万
- 35 岁父亲猝然病逝 (高血压一直没查), 母亲查出乳腺癌二期
- 38 岁妻子带走 23 万存款和最好的朋友远走, 他蹲在出租屋楼下点一根烟
- 临终前半年系统性写下悔录, 每条成为十八岁自己收到的凌晨短信

### 角色弧光
1. **卷一 (1-40)**: 被同班富二代小团体当众嘲讽 → 愤怒中收到第一条短信 → 完成第一个任务 → 初尝爽
2. **卷二 (41-85)**: 月考逆袭 → 主动选择不升班留原班 → 把原班 "改造" 成自己舒服的节奏
3. **卷三 (86-135)**: 运动会篮球爆种 → 校园风头正盛但冷处理 → 朋友圈成型 → 摄影副线开启
4. **卷四 (136-185)**: 给家里换空调冰箱 → 父亲 "血压高" 的小爆点 → 冬天给父亲买羊毛大衣
5. **卷五 (186-230)**: 寒假和青梅一起回老家 → 意识到 "陪伴" 比 "改变" 更重要
6. **卷六 (231-285)**: 多女主情感线同时升温, 陆辞选择对每个人真诚但不承诺
7. **卷七 (286-340)**: 志愿博弈, 他最终选择留在南安师大摄影系或冲同济, 留下悬念
8. **卷八+终章 (341-410)**: 高考 → 送别 → 开放结尾, 凌晨短信停了一个月, 直到某个大学宿舍的夜里, 它又亮了一次

## 青梅女主:苏知予 (Sū Zhīyǔ)

### 基础档案
- **年龄**: 18
- **身高**: 163 cm, 白皮, 腰细腿直, 江南女孩轮廓
- **外貌**: 大眼睛, 双眼皮深但眼尾微微吊, 笑起来两颗小虎牙; 常年一条马尾, 偶尔编成两股; 耳垂小, 容易红
- **班级**: 南安三中高三 (2) 班 (奥赛理科二班)
- **家庭**:
  - 父 **苏承砚** (48, 南安市中医院心内科副主任, 温和, 无框眼镜, 爱学习)
  - 母 **林语秋** (原武警总队医院护士, 2015 年肺腺癌去世, 知予十岁)
  - 单亲家庭, 父女感情深, 父亲近年常驻省会搞科研项目
  - 住安河小区 8 栋 501, 对门就是陆家

### 性格核心
- **表层**: 高冷 / 清淡 / 学霸 / 不好惹 / 对陌生人礼貌但不多话
- **深层**:
  - 极度傲娇, 嘴上狠心里软
  - 对陆辞这 "讨厌鬼" 刀子嘴, 但从小默默观察他
  - 怕老鼠 / 怕虫子 / 雷雨天不敢一个人睡
  - 爱吃甜食但要藏着吃 (怕发胖)
  - 对父亲的关心深藏不露, 父亲在外地她从不打电话, 但半夜常翻父亲朋友圈
  - 对陆家两口子 (陆叔陆姨) 视如亲人, 常在陆家蹭饭
- **口头禅**:
  - "滚啊!" "有毛病吧" "你死!" "我打你了"
  - 被夸到时: "哼。" (内心: 其实还挺开心的)
  - 气急时: "陆辞! 你! 我跟姨说去!"
- **怪癖**:
  - 爱看古言小说和东野圭吾
  - 自己练古筝, 不轻易演奏给人听
  - 抽屉里有一本日记, 几乎每一页都和陆辞有关, 但少有写他名字
  - 晒被子要晒在阳光最足那一面

### 技能 / 特长
- **学业**: 全年级前二十, 班内前五, 英语满分钉子户, 语文作文常被印成范文
- **古筝**: 十级, 小学四年级开始学, 偏古典
- **工笔淡彩**: 会画, 但只给自己看
- **烹饪**: 基本做不了, 会煮泡面加蛋

### 与男主的童年纠葛
- 幼儿园同班, 一起被接放学
- 小学二年级: 陆辞把她养的金鱼 "海葬" 在楼下泡桐树根, 被她追杀三条街
- 初一: 陆辞剪过她麻花辫尾一小截 (因她画丑了陆辞的速写), 她哭一下午, 第二天把陆辞作业本所有答案涂黑
- 初二: 她发烧, 陆辞不由分说公主抱她去医院, 导致她缺考数学; 陆辞那次得了人生唯一一次班级第一, 她记恨至今
- 高二下学期关系开始缓和, 高三仍互怼但底色亲近

### 角色弧光
1. **卷一 (1-40)**: 忽然发现陆辞不一样了 (早起买早餐 / 开始认真学习 / 不再去混小团体); 嘴上不信, 心里偷看
2. **卷二 (41-85)**: 陆辞月考爆种, 她第一次认真看他的试卷分数; 故意绕路和他同行回家
3. **卷三 (86-135)**: 看到他与其他女生的互动 (班花应采禾、文学社姜酌), 第一次体会 "吃醋", 嘴上更凶
4. **卷四 (136-185)**: 冬天, 她生理期趴桌上难受, 陆辞默默关上窗户; 她记了一整天
5. **卷五 (186-230)**: 寒假, 两家一起回青田村过年, 除夕夜一起放烟花; 独处戏最多
6. **卷六 (231-285)**: 意识到自己喜欢陆辞; 但不说, 因为 "他身边的人太多, 我不想只是其中一个"
7. **卷七 (286-340)**: 决定报复旦中文; 她主动帮陆辞补数学, 偶尔熬夜讲题到凌晨
8. **卷八 + 终章 (341-410)**: 高考后没告白; 送陆辞一本六年手账 (每一页她悄悄记的那些 "和陆辞有关" 的日常); 毕业晚宴跳一支舞后各自去不同城市; 结尾留给读者想象

### 关键写作提示
- **绝对不要**: 写她哭着求复合 / 写她 "为他忍让" / 写她戏份少于其他女主
- **关键意象**: 白色运动鞋 / 古筝琴弦 / 阳台晒被子 / 拉链上拉到下巴 / 雷雨夜敲对门 / 被晒红的耳垂 / 藏起的甜食包装纸
- **她的成长线**: 从 "好学生的骄傲" 到 "懂得在意一个具体的人"
- **语气训练**: 每一次她和陆辞对话, 先想她会说什么狠话, 再想她内心真实想法, 两者落差越大越好
