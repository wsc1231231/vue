# 01. 人物卡 — 男主与青梅

## 男主: 陆辞 (Lù Cí)

### 基础档案
- **年龄**: 18 岁
- **身高**: 182 cm, 瘦而有肩线, 长期打篮球锻炼的身材
- **外貌**: 眉骨高, 眼皮薄, 下颌线清晰, 笑起来有点邪气, 冷着脸时有距离感。头发黑而硬, 刘海常扫过眉毛。耳垂小, 不笑时常被误会凶
- **班级**: 临一中高三 (10) 班 → 首次月考后 → 高三 (3) 班 (奥赛三班)
- **家庭**:
  - 父 **陆卫东** (45岁, 青枫区青和五金建材店老板, 性子慢、不爱说话、爱抽烟)
  - 母 **许秀兰** (43岁, 临川市青枫区第二人民医院护士, 刀子嘴豆腐心, 唠叨)
  - 家境: 2017 年陆爷爷重病花光家底, 至今刚还清欠债, 月结余不多
  - 一家三口, 住青枫区 **安河小区** 6 栋 402, 对门 403 住苏家

### 性格核心
- **表层**: 吊儿郎当 / 嘴毒 / 对陌生人友善但带距离 / 懒散 / 对学习无所谓
- **深层**:
  - 对父母孝顺, 会替母亲拎菜, 会记住父亲鞋码
  - 对虚伪敏感, 被踩一次就永不回圈
  - 看似冷漠, 实际共情力强, 能看穿别人小心思
  - 务实, 信奉 "先解决自己的温饱, 再谈理想"
  - 对女生不主动, 但一旦接纳就极护短
- **口头禅与梗**:
  - "啧" "得了吧" "我超" "你妈的" (脏话不跨越屏幕)
  - "……行吧" "唉, 这日子" "我谢谢你"
  - 想装深沉被戳破时: "年轻人嘛, 有点想法很正常"
- **怪癖**:
  - 不喝甜豆浆
  - 吃饭必须热的
  - 睡前会把手机放到伸手够不到的地方以防熬夜
  - 对小孩子有耐心, 对大人没耐心

### 技能 / 特长
- **摄影**: 初二用压岁钱攒了一台二手富士 X-T10, 高中继续自学。擅长人像与街头扫街, 豆瓣账号 "光之弃子" 不到百粉, 风格偏阴翳系, 后期逐渐转向明亮记录
- **篮球**: 校队外援水平, 打 SG, 三分投得好, 关键球心脏大。连续一年长跑五公里, 体能很好
- **动手**: 会修小家电 (家里电器出故障都是他搞定)
- **做饭**: 会, 但不精 (区别于原型, 只做家常, 不以此立人设)

### 未来线 (三十八岁的陆辞)
- 从临川师专毕业, 干过工厂质检 / 外卖 / 摄影助理 / 短视频剪辑
- 28 岁相亲结婚, 32 岁创业做摄影工作室亏了 78 万
- 35 岁父亲病逝, 母亲查出癌症
- 38 岁妻子带走 23 万存款与最好朋友远走, 工作室倒闭, 签字离婚
- 蹲在出租屋楼下点烟, 回忆起高中被捧小团体羞辱的生日饭那晚
- 临终前半年系统性写下悔过录, 以 "回响" 形式回到十八岁自己脑中

### 角色弧光
1. **第一卷**: 被嘲讽后意识到自己在"讨好"; 系统激活; 初尝爽
2. **第二卷**: 依赖系统爆发学科天赋; 开始思考 "我要不要真的变优秀"
3. **第三卷**: 融入新圈子; 意识到系统只是助力, 真正的核心是自己
4. **第四卷**: 对家庭的责任感明确; 开始用赚的钱改造家里
5. **第五卷**: 乡土与青梅独处; 第一次对"陪伴"有清晰认识
6. **第六卷**: 面对多女主情感; 选择 **不给明确承诺** 但对每个人真诚
7. **第七卷**: 志愿填报博弈; 明白自己想要什么样的未来
8. **第八卷与终章**: 离别; 留下"回声未完"的开放结尾

## 青梅女主: 苏知予 (Sū Zhīyǔ)

### 基础档案
- **年龄**: 18 岁
- **身高**: 163 cm, 白皮, 腰细腿直, 标准江南女孩轮廓
- **外貌**: 大眼睛, 双眼皮深但眼尾微微吊, 笑起来有两颗小虎牙。常年一条马尾, 偶尔编成两股辫。耳垂小, 容易红。校服下可以看出胸线明显 (校园写实, 不色情化)
- **班级**: 临一中高三 (2) 班 (奥赛理科二班)
- **家庭**:
  - 父 **苏承砚** (48岁, 临川市中医院心内科副主任, 温和, 戴无框眼镜, 爱学习)
  - 母 **林语秋** (原武警总队医院护士, 2014 年肺腺癌去世, 知予十岁)
  - 单亲家庭, 父女感情深, 但父亲近年常驻省会搞科研项目
  - 住青枫区安河小区 6 栋 403, 对门是陆辞家

### 性格核心
- **表层**: 高冷 / 清淡 / 学霸 / 不好惹 / 对陌生人礼貌但不多话
- **深层**:
  - 极度傲娇, 嘴上狠心里软
  - 对陆辞这 "讨厌鬼" 刀子嘴, 但从小默默观察他
  - 怕老鼠 / 怕虫子 / 雷雨天不敢一个人睡
  - 爱吃甜食但要藏着吃 (怕发胖)
  - 对父亲的关心深藏不露, 父亲在外地她从不打电话, 但半夜常翻父亲朋友圈
  - 对陆家两口子 (陆叔陆姨) 视如亲人
- **口头禅**:
  - "滚啊!" "有毛病吧" "你死!" "我打你了"
  - 被夸到时: "哼" (内心: 其实我还是挺开心的)
  - 被陆辞气急时: "陆辞! 你! 我跟姨说去!"
- **怪癖**:
  - 看小说和漫画, 最爱古言与东野圭吾
  - 自己练古筝, 不轻易演奏给人听
  - 抽屉里藏着一本日记本, 内容很少有陆辞的名字但几乎每一页都和陆辞有关
  - 晒被子要晒在阳光最足的那面阳台

### 技能 / 特长
- **学业**: 全年级前二十, 奥赛二班前五, 英语满分钉子户, 语文作文常被印成范文
- **古筝**: 十级, 小学四年级开始学, 风格偏古典
- **绘画**: 会工笔淡彩, 但只画给自己, 不给人看
- **烹饪**: 基本做不了, 会煮泡面加蛋

### 与男主的童年纠葛
- 幼儿园同班, 一起被接放学
- 小学: 陆辞把知予最喜欢的金鱼埋在楼下泡桐树下 (自称 "海葬"), 被知予追杀三条街
- 初一: 陆辞剪过她的麻花辫尾一小截 (因她画丑陆辞的速写像), 知予哭了一下午, 第二天把陆辞作业本上所有答案涂成黑色
- 初二: 知予发烧被陆辞不由分说公主抱去医院, 缺考数学, 陆辞得了人生中唯一一次班级第一, 知予记恨至今
- 高二下学期开始缓和, 高三仍互怼但底色是亲近

### 角色弧光
1. **第一卷 (1-40章)**: 忽然发现陆辞变了 (早起买早餐 / 月考爆种 / 不再去混圈); 嘴上不信, 心里偷看
2. **第二卷 (41-90)**: 陆辞升班后离她更近; 每天回家路同行; 她开始默默关心他的成绩
3. **第三卷 (91-150)**: 目睹陆辞与其他女生的相处 (苏晚、江栀 等); 第一次发现自己会 "吃醋", 但嘴上更凶
4. **第四卷 (151-200)**: 大联考爆发期, 两人偶尔熬夜通电话讲题; 第一次主动在雷雨夜敲他家门
5. **第五卷 (201-240)**: 寒假, 两家一起回青山镇 (父亲送她去陆家过年); 独处戏最多, 感情质变
6. **第六卷 (241-290)**: 意识到自己喜欢陆辞; 但不说, 因为 "他身边太多人, 我不想变成其中一个"
7. **第七卷 (291-340)**: 冲刺期; 各自志愿博弈, 她选择复旦, 他可能去隔壁同济或留临川
8. **第八卷与终章 (341-400)**: 高考后; 她没有告白, 而是送给陆辞一本笔记本 (里面是六年来他出现的每个重要日子); 最终开放结尾, 她去了上海, 陆辞是否跟去, 留给读者想象

### 关键写作提示
- **绝对避免**: 写她哭着求复合 / 写她 "忍让" / 写她的存在感低于其他女主。她是唯一贯穿全书的第一女主, 即使结局开放, 笔墨要均衡
- **关键意象**: 白色运动鞋 / 古筝琴弦 / 阳台晒被子 / 拉链上拉到下巴 / 雷雨夜敲门 / 她晒红的耳垂
- **她的成长**: 从 "好学生的骄傲" 到 "懂得在意别人"
