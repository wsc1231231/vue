## 下游 AI 创作使用说明

1. **先读 00-05 全部设定类文件**, 建立人物 / 规则 / 文风认知
2. **按卷读大纲**, 一次只读一卷, 读完该卷再开始写
3. **每一章写作前的检查清单:**
   - 本章大纲节点
   - 涉及人物卡 (语气 / 口头禅 / 情感状态)
   - 风格工具包 (梗 / 句式 / 禁忌)
4. **每章产出 2800-3500 字**, 严格按大纲推进, 不跳节点, 不合并
5. **不可改动的节点** (告白 / 争吵 / 分离 / 升班 / 系统任务结算): 原样保留; **过渡桥段** (日常对话 / 跑操 / 食堂) 可自由发挥
6. **每章结尾必须有钩子** (悬念 / 冷笑话 / 暧昧瞬间 / 下章预告式动作)
