# 第 4 章 · 第一行代码

第二天上午九点。

江叙在餐桌前写代码。

笔记本屏幕上是 Notepad++，打开一个 .html 文件。他昨晚睡前打到第 30 行。早饭完又坐回来。

他在做一个小东西——县医院消化内科的"值班表"网页。

2014 年的县医院还用纸质排班表，每周手写一份贴科室门口。苏主任前一阵子来江家喝酒时吐槽过一次——"每周折腾这么一张纸，字写歪了还得重来。"

男主那时候听过。这次用上。

他想的系统很简单：科室可以自己改，每周自动生成一张，值班人员的照片能挂上去，家属手机也能看。

前端 HTML + CSS + JS，后端 PHP + MySQL，XAMPP 本地跑就够演示。

HTML 骨架昨晚搭好了，现在在写 CSS。

——

"咚咚咚。"

敲门声。

江叙没抬头："进来。"

门开了。苏晚探进半个头。

"你家门没锁。"

"就等你。"

"……"

她进来换鞋，走到餐桌旁站着。

"你在写啥？"

"代码。"

"……我看看。"

她拉一把椅子坐他旁边，手撑在桌沿。

屏幕上一堆看不懂的英文字母加大括号加分号。

"……"

"你读得懂？"江叙问。

"读得懂读不懂关你什么事。"

他笑了一下，继续敲键盘。

——

她坐旁边看了十分钟。

"这是什么？"

"CSS。"

"英文？"

"英文。"

"……你初中学的？"

"嗯。"

"……"

她没再说。

但她看他的手在键盘上"哒哒哒"敲——一行 CSS 五秒钟搞定，补齐所有括号和分号，跟读课文一样顺。

——

敲了半小时，江叙停下保存。

"饿了。"

"有点。"

"中午我下面。"

"又下面。"

"你有意见？"

"……没。"

他起身进厨房。

苏晚坐原地，盯屏幕。伸手轻轻按一下键盘。

屏幕没反应。

"江叙！"

"干嘛？"

"你这键盘不好使。"

"我没关机，你按的是 Esc 键。"

"……"

她把手缩回去。

——

面端上来。两碗清汤面两个煎蛋。

边吃边聊。

"江叙。"

"嗯？"

"你写这个卖给谁？"

"先送给你爸。"

苏晚的筷子停住："……啥？"

"我做一个值班表系统，免费给你爸看看。要是他觉得好用，就让他跟院里其他科室推荐一下。"

"……你图什么？"

"让他先见见我会什么。以后医院要做东西，他第一个想到我。"

"……"

"算是敲门砖。"

苏晚扒完那口面，放下筷子："你这脑子怎么转这么快？"

"……"

"前几天你连我爸干啥都说不清。"

江叙："我最近在琢磨。"

"琢磨什么？"

"你以后就知道了。"

"你又来这套。"

"嗯。"

——

吃完苏晚帮他把碗洗了。

她没说要走。又坐回餐桌边看他写代码。

屋里空调"呼呼"吹，窗外阳光烈。

江叙左手边笔记本，右手边苏晚。

他没抬头，但他知道她就在旁边。

——

下午三点，HTML + CSS 的演示页面做完。他打开 Chrome 预览。

屏幕上出现一个干干净净的网页：

> 云溪县人民医院 · 消化内科 · 本周值班表
>
> 周一 · 苏建军主任 · 早班
> 周二 · 李医生 · 早班
> 周三 · 王医生 · 早班
> ……

他保存打包。

"苏晚。"

"嗯？"

"今晚去你家一趟，我跟你爸说一声。"

"嗯。"

她没问什么，眼睛还盯屏幕。

——

傍晚六点半。苏主任在客厅看新闻联播前的广告。

江叙拎着笔记本过去。

"苏叔。"

"哎小叙。这大晚上的找我有事？"

"给您看个东西。"

他把笔记本搁茶几上，开机，双击 Chrome。

页面打开。

苏主任把遥控器放下凑过来。

"……这是啥？"

"你们科室的值班表。我做了个网页版。"

"网页版？"

"每周自动出一张，能改能加照片。你手机里也能看。"

苏主任眯眼："……你做的？"

"我做的。"

"怎么做的？"

"写代码。昨天买了个二手电脑，晚上到两点写的。"

"……"

"苏叔你看看，要是觉得好用，就给你们科室用用。不要钱。"

"不要钱？"

"不要。就想让您看看我会这个。"

苏主任盯屏幕看了十秒。

"……小叙，你这是要做生意？"

"嗯。"

"给医院做生意？"

"不只医院。哪儿要，哪儿做。"

"……"

"您这边要是哪个科室朋友要用，您帮我递个话。"

"……"

苏主任抬头看他，笑了一下。

"你小子，前几天还跟刘飞鹏那帮混，今天突然就要开公司了？"

"没开公司，就我一个人。"

"你一个人做得过来？"

江叙看一眼旁边站着的苏晚，又转回看苏主任。

"苏晚帮我。"

苏晚："……啥？"

苏主任笑出声："嗯，挺好。挺好。"

苏晚："爸你笑啥！"

"我还没说啥呢。"

苏晚："……"

——

江叙把笔记本关掉，收进包里。

"苏叔，我就问一句。"

"嗯？"

"你们科室有人手机号我能登记吗？后面做的时候可能要问他们几个问题。"

苏主任想了两秒。

"先给你一个——我的。有啥先问我。"

"行，谢谢叔。"

——

江叙拎着电脑走。

苏晚送到门口。

"江叙。"

"嗯？"

"……你帮我帮你了？"

"你还没帮。等后面真要做项目，你帮。"

"……哼。"

她关门。

江叙笑了一下，开自家门。
